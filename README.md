# Great Library

Build your personal AI-powered knowledge base. Upload documents and ask questions about them using Google's File Search
API.

## Features

- **Upload Files** - Add documents to your searchable library
- **List Files** - Browse and manage your uploaded documents
- **Quick Note** - Capture text snippets or clipboard content as notes
- **Ask the Library** - Query your documents in Raycast AI Chat and get answers with citations

## Setup

1. Get a Google AI API key from [Google AI Studio](https://aistudio.google.com/api-keys)
2. Enter your API key in the extension preferences
